6. How to use a clone Xbox controller on Linux

I bought an ELE-GATE Xbox One controller a while ago, which I thought it was going to work out of the box with my PC, to no one’s surprise, it didn’t work, so I kept it in a drawer.

Figure 1: Looks cool, it didn’t cost me much ($499 pesos)

6.1. Until recently

I found a GitHub repository that added the necessary drivers for Xbox controllers to work, so I gave it a try.

6.2. Didn’t work as expected

My controller wasn’t getting recognized by the PC, so I thought it was the fact that I was using a clone controller. Looking at the issues page there was a thread about a similar thing.

6.3. I was coming to my solution

[https://github.com/paroj/xpad/issues/161#issuecomment-855162400][this comment forked the fix] that the thread was talking about, so I went to [https://github.com/sirkhancision/xpad][sirkhancision’s] repo to download the specific model driver.
